# **Daily Planner Layout**



A static daily planner layout created using **HTML** and **CSS**.  

This project focuses only on ***structure and visual layout***. Users cannot type or interact with the planner.



#### **## Overview**

This project demonstrates basic front-end concepts such as ***HTML tables*** and ***CSS styling***.  

It does not include JavaScript or form inputs, so the planner is for display purposes only.



#### **## Features**

\- Static daily planner table

\- Days displayed as columns

\- Time slots shown as rows

\- Clean and simple layout



#### **## Technologies Used**

\- HTML5

\- CSS3



#### **## Project Structure**

**├── index.html**

**├── style.css**

**└── README.md**





#### **## How to View**

1\. Download or clone the repository  

2\. Open `index.html` in any web browser  



No installation or setup required.



#### **## Learning Purpose**



This project was created to practice:

\- HTML table structure

\- CSS styling

\- Linking external CSS files



#### **## Limitations**

\- Not responsive

\- No user input or interactivity

\- Tasks cannot be typed or saved
